Un compañero virtual que acompaña al estudio al acompañar al usuario en sesiones de estudio. Las sesiones de estudio se componen de bloques de estudio seguidos de bloques de pausa intercalados, ambos de duración configurable por el usuario. Durante la sesiones, el robot muestra diferentes animaciones amigables para acompañar al usuario, siendo capaz de detectar y reaccionar si el usuario abandona el escritorio por un periodo prolongado. 

Requerimientos de hardware
- Pantalla 8x8
- Pantalla LED 2x16
- 3 botones
- Sensor de proximidad ultrasónico
- Protoboard
- Arduino Uno
